CodeAlpha - World Population Analysis

Project Overview

The project demonstrates the complete data analytics workflow, including:

 Web Scraping
 Data Cleaning
 Exploratory Data Analysis (EDA)
 Data Visualization
 Dashboard Creation in Microsoft Excel

The dataset was collected from Wikipedia using Python and analyzed to discover population trends across countries.

 Objectives

- Collect real-world population data from the web
- Clean and prepare the dataset
- Perform Exploratory Data Analysis (EDA)
- Create meaningful visualizations
- Build an interactive Excel dashboard
- Present insights that support data-driven decision making

Task 1 - Web Scraping 

Dataset is scrapped from Wikipedia using Python.

Steps Performed -

1. Read the HTML tables using Pandas

2. Retrieved the population table only

3. Dropped unnecessary rows

4. Cleaned the Dataset

5. Exported cleaned data as CSV file

-------------------------------------------------------------------------

Task 2 - Exploratory Data Analysis

Performed EDA on the dataset to find answers to various analytical questions.

Analysis included -

1. Dataset Info

2. Check for Missing Values

3. Check for Duplicates

4. Check Datatypes

5. Statistical Summary

6. Country with Highest Population

7. Country with Lowest Population

8. Top 10 Populated Countries

9. Countries with Population > 100 Million

10. Outlier Analysis

11. Distribution of Data

12. Population Statistics

-------------------------------------------------------------------------

Task 3 - Data Visualization 

Visualizations were created using matplotlib and seaborn library.

Charts included -

1. Top 10 Countries by Population

2. Population Distribution Histogram

3. Horizontal Population Bar Chart

4. Population Share Pie Chart

5. Box Plot - Outlier Analysis

Interactive Dashboard was also created in excel using Pivot Tables and Pivot Charts.
